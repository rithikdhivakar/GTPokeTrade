About PokéTrade

PokéTrade is a fun, interactive card trading app built using Django and the Pokémon TCG API. You can collect cards, propose trades, earn achievements, and build your ultimate collection!

Tech Stack:

  • Python + Django
  • HTML5, CSS3
  • SQLite DB
  • Pokémon TCG API
  • Optional: PokeAPI integration
